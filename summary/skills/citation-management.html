用户故事

作为一名医学写作人员 / 学术研究支持人员，我希望在参考文献清洗与投稿前整理场景中输入杂乱的参考文献列表、已有 DOI、arXiv ID 或不完整书目信息，让 AI 帮我生成统一格式的参考文献条目、补全缺失 DOI、提取 arXiv 信息并标出疑似重复或信息不完整的引用，以便我能更快整理参考文献库 / 支持论文写作提交 / 做学术信息汇总。

输出文件

一、最终交付物（1 个）

  • skills_output/引用管理/references.bib：格式清洗、去重并排序后的正式 BibTeX 文件，包含全部 13 条引用，可直接用于 LaTeX 文档

二、中间工作文件（5 个）

  • skills_output/引用管理/dois.txt：从参考文献中识别出的 4 个 DOI
  • skills_output/引用管理/arxiv_ids.txt：从参考文献中识别出的 5 个 arXiv ID
  • skills_output/引用管理/doi_entries.bib：通过 CrossRef API 从 DOI 提取的原始 BibTeX
  • skills_output/引用管理/arxiv_entries.bib：通过 arXiv API 从 arXiv ID 提取的原始 BibTeX
  • skills_output/引用管理/references_raw.bib：将 DOI 条目、arXiv 条目及手动编写条目拼接后的原始合并文件

三、补充处理文件（2 个）

  • skills_output/引用管理/remaining_arxiv.txt：第一次提取后补录的 arXiv ID
  • skills_output/引用管理/remaining_arxiv_entries.bib：对补录 arXiv ID 提取的 BibTeX 条目
